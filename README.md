# Movie Go App
 
## Features
-**Log in or sign up to access books**

-**Browse different movies**

-**Search for a specific movie**

-**Dark and light mode**

-**English and Arabic languages**

-**Watch movie videos**

with Lottie splash screen 

## This is done using:-
- **MVVM Architecture** 

- **Cubit State Management** 

- **Dio  with API Integration** 
- **Shared preferences** 
- **Responsive Design**
- **Localization**
- **Flutter offline**
- **youtube player**
